### linux命令
+ 查看文件内容
  + `ls`：查看当前目录下的文件
  + `ls -a`：查看当前目录下的文件，包括隐藏文件
  + `ll`: 列表格式查看当前目录下的文件
  + `cat`：查看文件内容
  + `more`：查看文件内容，一页一页的查看
  + `less`：查看文件内容，一页一页的查看，可以向上翻页
  + `head`：查看文件前几行
  + `tail`：查看文件后几行
  + `grep "xxx" aaa.js`：查找文件内容,`xxx`为要查找的内容，`aaa.js`为要查找的文件
+ 文件操作
  + `touch`：创建文件
  + `mkdir`：创建文件夹
  + `rm`：删除文件
  + `rm -r`：删除文件夹
  + `rm -rf`：强制删除文件夹，不提示
  + `mv xxx`：移动文件
  + `mv xxx yyy`：重命名文件
  + `cp a.js a1.js`：复制文件,`a.js`为原文件，`a1.js`为复制后的文件
  + `cp -r a b`：复制文件夹， `a`为原文件夹，`b`为复制后的文件夹
  + `vim`：编辑文件
  + `vimtutor`：vim教程

+ 登陆
  + `ssh work@服务器地址`：登陆远程服务器
